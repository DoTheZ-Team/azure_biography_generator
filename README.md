# biography_generator

> 가천대학교 Azure software workshop 팀프로젝트 결과물

### 주제설명
사용자 이름, 나이, 꿈을 말해주면 사용자의 위인전을 생성해주는 서비스

### 사용 기술
GPT 3.5 API, Streamlit, langchain
